Buttons

  • ● Click 'Build' button to start fractal drawing based on your input information
  • ● Click 'Export' button to download the current fractal

Buttons

Inputs

  • ● Enter input information before clicking 'Build'
  • ● Alert messages will inform you of valid values for input fields
  • ● Drag 'Scale' to change the scale of your fractal image
  • ● Use 'Color Theme' input to choose a theme of four colours for Newton fractal. The 'From picture' option creates a pallette of four colors from given picture using median cut quantization
  • ● Use 'Color' input to choose a color for Vicsek fractal
